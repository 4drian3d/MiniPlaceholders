# MiniPlaceholders
MiniMessage Component-based Placeholders for PaperMC and Velocity platforms

## Commands:
### Velocity
`/vminiplaceholders <parse:help> <me:player> <player-to-parse>`
### Paper
`/miniplaceholders <parse:help> <me:player> <player-to-parse>`
